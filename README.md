# Code Deployment Pipeline Tracker

A Spring Boot DevSecOps-style pipeline tracker system.

## Features
- Submit code changes
- Scan workflow
- Approval workflow
- Deployment tracking
- Rollback support
- Role-based access
- Swagger API documentation
- MySQL integration
- JUnit + Mockito tests

## Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- Swagger/OpenAPI
- Mockito
- Maven
